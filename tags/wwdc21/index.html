What's new in Swift 5.5?

Swift 5.5, available in any Xcode 13 beta, includes a huge number of changes to the language. At WWDC this year, a lot of these changes were covered by session videos. However, despite being published before WWDC, this in-depth article from Paul Hudson is still worth a read. Paul walks through most of the language changes introduced in Swift 5.5, with code samples for each. Lot's to cover here.

AsyncImage in SwiftUI

New in iOS 15 is AsyncImage, a new SwiftUI View for fetching and displaying remote images from a URL. Here, Gabriel Theodoropoulos demonstrates how to use this new type, and highlights some of its pros and cons.

What is new in SwiftUI after WWDC21

SwiftUI took another substantial and welcomed step forward this year at WWDC. Along with some expected improvements, such as the new refreshable ViewModifier, we also saw some less anticipated additions such as the new TimeLine View type. In this post, Majid Jabrayilov provides his summary of significant SwiftUI additions announced at WWDC21.

Core Store

Core Data is an object graph and persistence framework provided by Apple. Introduced in iOS 3.0, it's still relevant but is also generally acknowledged as not being particularly Swift friendly. In this post, the excellent Daniel Steinberg makes an early prediction for a Swift replacement of Core Data which he thinks might be unveiled at this year's WWDC.
