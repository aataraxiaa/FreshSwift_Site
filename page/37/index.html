A Quick Tip For String Performance

When we create Swift strings, they are backed by either the Swift String type, or the Objective-C NSString type. This quick tip shows us how to improve performance by forcing our declared strings to be back by the Swift String type.

#string  #performance 

Swift Evolution moves to Discourse

Swift Evolution has up until now used mailing lists for all community discussions. While mailing lists served the purpose adequately, they were not ideal. Now, the move to Discourse is under way. Discourse provides a more modern, forum-based solution, and it’s where all Swift community discussions will now happen.
